7.1) Пример 1:
	notExcluded - included 
	// флаг, говорящий о том, включена ли страница чертежа в отображаемый список
     Пример 2: 
	shouldHideFlag - shouldHide
	// флаг, говорящий о том, что элемент должен быть скрыт
     Пример 3:
	flag - neededValueFound
	// флаг, говорящий о том, что нужное значение найдено и цикл необходимо прервать
     Пример 4: 
	createdFlag - created
	// флаг, говорящий о том, что модель была успешно создана
     Пример 5:
	notArchived - archived
	// флаг, говорящий о том, помещена ли текущая задача в архив


7.2) done - можно использовать для системы задач - когда задача выполнена, устанавливать значение как true.

     success - можно использовать, когда pdf файл чертежа успешно разложился на svg картинки.

     found - при переборе некоторого списка с элементами, устанавливать его как true, 
	     если в списке найдется элемент с нужным идентификатором.

     error - хорошо подойдет, если шлюз для устройств не смог установить 
	     соединение с iot-платформой.

7.3) Пример 1:
	e - existPanel
	// переменная внутри цикла, который перебирает использующиеся в данный момент панели

7.4) Пример 1: 
	master/slave - переменные присваиваются панелям с информацией на экране, 
	где master - панель, размер которой пользователь меняет руками, 
		     а slave - панель, размер которой подстроится под размер панели, помеченной как master.
	
     Пример 2: 
	parent/children - переменные, хранящие в себе данные о родительском и дочернем элементе/элементах
		          (имеются ввиду элементы ifc модели).
	

7.5) Пример 1:
	tenantId - от этой переменной можно вообще избавиться 
		   и присваивать значение полю объекта без использования промежуточной переменной.

     Пример 2: 
	id - deviceProtocolId
	id - deviceModelId
	id - deviceProducerId
	// все три переменных (объекты одного и того же класса, хранящего идентификационные данные)
	// потом попадают в нужные им объекты - для наглядности добавил префикс, 
	// явно говорящий о том, в какой объект они попадут.
